Politique de Confidentialité

Dernière mise à jour : Avril 2025

1. Introduction

Bienvenue dans Check-it, une application développée pour aider les commerçants à vérifier si un numéro de téléphone a été signalé comme frauduleux par d'autres utilisateurs. Cette politique explique comment nous traitons les données personnelles dans notre application.

2. Informations collectées

Nous collectons les numéros de téléphone soumis volontairement par les utilisateurs lorsqu’ils souhaitent vérifier ou signaler un contact. Aucune donnée sensible, comme votre nom, adresse ou contacts personnels, n’est collectée sans votre consentement explicite.

3. Utilisation des données

Les données sont utilisées uniquement pour :
- Vérifier si un numéro a été signalé
- Afficher l’historique des signalements associés à un numéro
- Aider les commerçants à éviter les fraudes

4. Partage des données

Les numéros signalés sont partagés de manière anonyme avec les autres utilisateurs de Check-it. Nous ne partageons aucune donnée avec des tiers commerciaux ou publicitaires.

5. Sécurité

Nous prenons des mesures de sécurité techniques et organisationnelles pour protéger les données stockées et transmises via l'application.

6. Vos droits

Vous pouvez demander la suppression d’un numéro signalé par erreur en nous contactant à : oran.inturk@gmail.com.
